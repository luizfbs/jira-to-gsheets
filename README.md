# JIRA to Google Sheets (jira-to-gsheets)
> Extract data from JIRA Cloud and save to Google Sheets.

![](screenshot.png)

## Installation

```sh
cd ./path/to/jira-to-gsheets
npm install
```

## Usage example

```sh
npm start
```
